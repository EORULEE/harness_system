# disclosure-and-integrity-rules.md — 공개·정직성 규칙

> 모든 Writing Suite 산출물의 메타에 적용. 글로벌 "허위 준수 금지"의 글쓰기판.

## 1. 정직 공개 (메타 필수)
산출물 메타에 다음을 명시한다:
- 사용한 writer role·실제 사용 엔진(pro/flash/폴백 — 위임 에이전트 메타에서).
- 재작성 반복 횟수.
- **[확인 필요]** 항목 수(미확인 사실·수치·인용).
- 적대검토 수행 여부(미수행 시 "환경 제약으로 미수행" 사유 1줄).

## 2. 금지
- 안 한 검토를 한 것처럼 표기 금지(메타↔실제 일치).
- AI 생성 사실을 검증된 사실로 위장 금지.
- 과장·홍보성 단정("최초의", "획기적인") 근거 없이 금지.

## 3. AI 작성 고지
- 요청 맥락(논문·제안서 등)에서 AI 보조 작성이 정책상 고지 대상이면 사용자에게 안내.
- 표절·자기표절 위험 항목은 경고.

## 4. secret 보호
- API key·token·자격 원문을 산출물·로그·메타 어디에도 출력 금지(AC19).
- 자격은 **파일 이름만** 언급(예: `gemini.env`).

## 5. 권위
- 본 규칙 위반 의심 시 stop-guard·hookify가 최종 권위. 충돌 시 안전측.
